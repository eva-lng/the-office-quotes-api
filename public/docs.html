API Documentation

Welcome to the Office Quotes API! This API provides access to a collection of quotes from the popular TV show "The Office." You can retrieve random quotes or quotes from specific characters. Authentication is not required to access the API.

How to use the API

To fetch a quote, simply make a GET request to the respective endpoint. Response is returned as a JSON object.

Example

Request:
GET /api/random
Response:
{ "character": "Michael Scott", "episodeName": "Diversity Day", "episodeNumber": 2, "id": 5, "season": 1, "text": "Abraham Lincoln once said that, 'If you're a racist, I will attack you with the North.' And those are the principles that I carry with me in the workplace." }

Endpoints

Fetch a random quote from any character

GET /api/random

Fetch a random quote from a specific character

GET /api/random/{char} replace {char} with character's first name

Fetch all quotes from a specific character

GET /api/all/{char} replace {char} with character's first name
